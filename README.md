# Geospatial Valuation

This workspace contains a starter structure for a geospatial valuation project.

## Project structure
- data/raw/ - raw input data
- data/processed/ - cleaned and prepared data
- models/ - trained models or checkpoints
- notebooks/ - exploratory analysis notebooks

## Suggested workflow
1. Place raw datasets in data/raw/
2. Create preprocessing and modeling scripts in the project root or notebooks/
3. Save trained artifacts in models/
4. Keep generated outputs out of version control via .gitignore
